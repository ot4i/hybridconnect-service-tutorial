Import projects

When you click Import, the PrimeNumbers application will be created in you workspace.

PrimNumbers uses the Compute node to look for all the prime numbers between 0 and 100000. This will be use as a way to create some data for the Product Insights service dashboard to show.

Getting an API host and API key

Before you can register your integration server with the Product Insights service you will need to have an API host and API key for you Product Insights service.

If you already have one then you can move on to the Run section.

You can follow the instruction in Configuring the IBM Cloud Product Insights service of the IBM Knowledge Center to get the API host and API key.

Follow these steps to complete the tutorial

Follow the instruction to register IBM Integration Bus with Product Insights service.

  1. Open an IBM Integration Console and use the command mqsichangebluemixreporting to register you integration server. Instruction can be find in Configuring an integration server to send startup and usage information to IBM Cloud Product Insights of the IBM Knowledge Center.
  2. After registration, goto your web browser to logon to your IBM Bluemix account to open the Integrate services dashboard and select the Integrate Service you just created.
  3. In the dashboard, select the IBM Integration Bus.
  4. If you registered in a group then your integration node with be a member of that group, expand the group.
  5. Select your integration node in the left column and in the center column select your integration server. The right-hand column will show the usage data for the selected integration server.
  6. On the right-hand column, if Usage is not selected, select it.
  7. Under Usage the chart can select to show one of three measurements. CPU time shows the CPU usage time by the selected server (in milli-seconds). Number of active CPUs shows the number of active CPUs available for use. Resident set size shows the maximum recorded memory usage.
  8. Goto your IBM Integration Toolkit and open the PrimeNumbers.msgflow and run it in the Flow Exerciser with an empty message.
  9. Goto your web browser and refresh the dashboard to see the usage data. Depending on the options used in the registation step, it might take a few minutes for the data to be update.

This tutorial has shown how to register and monitor the usage of IBM Integration Bus with Hybrid Connect service.
